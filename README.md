# clippy-alert

When you miss Clippy.

## Usage

```
clippy-alert "My Text Here"
```

This will send a clippy-based alert.

## Building

* Clone repo
* go get -u github.com/gobuffalo/packr/v2/packr2
* packr2 build
* packr2 install

Assumes Go 1.11 with GO111MODULE=on or Go 1.12.
